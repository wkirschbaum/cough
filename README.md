# Cough
Short description and motivation.

## Usage
You just have to mount this rails engine in your routes file
```
mount Cough::Engine => '/cough'
```

or to any other path your prefer.

## Installation
Add this line to your application's Gemfile:

```ruby
gem 'cough'
```
